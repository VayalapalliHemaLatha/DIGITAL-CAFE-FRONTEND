# Cafe images

Drop your images here to use as backgrounds:

- `hero.jpg` – home hero (e.g. brunch/cafe table)
- `about.jpg` – about section
- `login.jpg` – login page header
- `register.jpg` – register page header

Then update the image URLs in `src/pages/HomePage.js`, `LoginPage.js`, and `RegisterPage.js` to use `/images/hero.jpg`, etc.
